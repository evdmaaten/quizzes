VWO 5 Scheikunde – interactieve quizset

Deze set bevat 25 quizzen op VWO 5 toetsniveau. Elke quiz kiest willekeurig 10 vragen uit een vraagbank, slaat je beste score op in de browser op, werkt goed op mobiel en tekent waar relevant eenvoudige grafieken automatisch.

Upload alle bestanden naar je GitHub Pages repository. Dit bestand moet index.html heten.
